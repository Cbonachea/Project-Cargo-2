# Project-Cargo 2

 2D Physics based spaceship landing that started it's life as a Lunar Lander (Atari '79) clone.
 This project was a great way to learn to implement rapid prototyping, Agile techniques, and the Unity environment.
 Live streaming the development of this game on Twitch lead to the beggining of a small indipendant games studio 
 called Spirit Quest Softworks cofounded by a long time college friend.
